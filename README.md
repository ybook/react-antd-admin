# react-antd-admin

A simple and complete front-end management solution based on Ant Design

## Usage

1. Clone project code.

```bash
git clone https://github.com/shx996/react-antd-admin.git
cd react-antd-admin
```

2. Installation dependence.

```bash
yarn install
```

Or

```bash
npm install
```

3. Start local server.

```bash
npm run start
```
